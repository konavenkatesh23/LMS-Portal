# Advanced LMS Portal

A modern Learning Management System (LMS) built using React.js, Vite, Tailwind CSS, and React Router.

## Features

- User Authentication
- Dashboard
- My Courses
- Course Details
- Video Learning
- Live Classes
- Assignments
- Certificates
- Student Profile
- Settings
- Responsive Design

## Tech Stack

- React.js
- Vite
- Tailwind CSS
- React Router DOM
- JavaScript (ES6)
- React Icons
- jsPDF

## Installation

```bash
git clone https://github.com/konavenkatesh23/LMS-Portal.git
cd LMS-Portal
npm install
npm run dev
```
